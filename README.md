# Gemini-Chatbot-System
Gemini Chatbot System is an AI-powered document analysis tool built using Python and the Google Gemini API. It supports PDF, JSON, TXT, and CSV files for summarization and question answering. The project uses Flask for backend processing and Streamlit for an interactive interface. It showcases skills in NLP, API integration, and data handling.
# 🤖 Gemini Chatbot System

## 📌 Description
Gemini Chatbot System is an AI-powered document analysis tool built using Python and the Google Gemini API. It allows users to upload files and get summaries or answers based on the content.

## ❓ Problem Statement
Reading and analyzing large documents is time-consuming. Users need a quick and efficient way to extract meaningful insights from different file formats.

## 💡 Solution
This system processes documents like PDF, JSON, TXT, and CSV and uses Gemini AI to generate summaries or answer user queries. It combines backend processing with an interactive UI.

## 🛠️ Tools & Technologies
- Python  
- Flask  
- Streamlit  
- Pandas  
- Google Generative AI (Gemini API)  
- dotenv  

## 🚀 Features
- Supports multiple file formats (PDF, JSON, TXT, CSV)  
- AI-based summarization and Q&A  
- Interactive UI using Streamlit  
- Backend API using Flask  
- Automatic file type detection
